**使用SPMS-codeblocks文件**
### 配置easyx 
下载库文件   https://codebus.cn/bestans/easyx-for-mingw  
解压缩，将 include 文件夹下的 easyx.h 和 graphics.h 拷贝到 D:\App\CodeBlocks\MinGW\x86_64-w64-mingw32\include\ 文件夹里<b>（根据你按照codeblocks的位置）</b><br>
